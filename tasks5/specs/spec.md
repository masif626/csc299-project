# Specification: MindQuest CLI

## 1. Overview
Mind Quest is a command line application in Python that merges a personal journal with a task manager.

## 2. Core Features
- **Quests** Add, list and complete tasks stored in SOLite database.
- **Journal** Add daily journal entries.

## 3. Database
- Uses 'sqlite3' with a 'mindquest.db' file.
- Tables: 'quests' and 'journals'.
