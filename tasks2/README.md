# Tasks 2 Project : Using SQLite database to store tasks.

## How to Run
1. Make sure Python is installed
2. Go to 'tasks2' directory in terminal.
3. The program uses database file called "tasks.db".

### List all tasks
py tasks.py list

### Add new task
py tasks.py add "Your new task here"

### Search for tasks
py tasks.py search "search term"
